# Trading-App
A simple Robinhood clone, it's an investing application that allows users to purchase stocks in publicly traded companies.

# Tech Stack
<ul>
  <li><b>Frontend:</b> React</li>
  <li><b>Backend:</b> Firebase</li>
  <li>Finn Hub Api</li>
  <li>React chartsjs</li>
  <li>CSS</li>
</ul>
